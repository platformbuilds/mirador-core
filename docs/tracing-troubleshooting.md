# Tracing Troubleshooting Guide

## Overview

This guide provides troubleshooting steps for distributed tracing issues in MIRADOR-CORE.

## Common Issues

### Missing Traces

**Symptoms:**
- Traces not appearing in queries
- Incomplete trace data

**Resolution:**
1. Verify tracing configuration
2. Check backend connectivity
3. Review trace ingestion logs

### Trace Performance Issues

**Symptoms:**
- Slow trace queries
- High latency in trace operations

**Resolution:**
1. Check trace storage performance
2. Review query patterns
3. Optimize trace retention policies
